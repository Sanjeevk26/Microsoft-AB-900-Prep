# Introduction to Microsoft Purview

## Overview

Microsoft Purview is a unified suite of solutions used to:

- Govern data  
- Protect data  
- Manage risk and compliance  

It helps organizations understand, secure, and manage their data across Microsoft 365 and other environments.

---

## Core Areas of Microsoft Purview

Microsoft Purview focuses on three major areas:

1. Data Security  
2. Data Governance  
3. Risk and Compliance  

---

## Microsoft Purview Solutions

### 1. Data Security

Includes:

- Information Protection  
- Data Loss Prevention (DLP)  
- Insider Risk Management  
- Copilot capabilities within Microsoft Purview  

---

### 2. Data Governance

Includes:

- Data Governance solution  

Used for understanding and managing organizational data.

---

### 3. Risk and Compliance

Includes:

- Audit  
- eDiscovery  
- Data Lifecycle Management  
- Compliance Manager  
- Communication Compliance  
- Copilot capabilities within Microsoft Purview  

---

## Additional Solution

### Microsoft Purview DSPM for AI

DSPM = Data Security Posture Management

Used for:
- Monitoring and securing AI-related data usage  
- Managing AI data exposure and risk

# Microsoft Purview Information Protection

## Overview

Microsoft Purview Information Protection focuses on three major goals:

1. Know your data  
2. Protect your data  
3. Prevent data loss  

---

# 1. Know Your Data

Helps identify and classify sensitive information.

---

## Sensitive Information Types (SITs)

Built-in or custom rules used to identify sensitive data.

Examples:
- Credit card numbers  
- Passport numbers  
- Tax IDs  
- Medical information  

---

## Trainable Classifiers

Used to classify content based on examples and machine learning.

Examples:
- Contracts  
- Resumes  
- Financial documents  

---

## Data Classification Explorers

### Data Explorer
Shows locations containing:
- Sensitive Information Types  
- Sensitivity labels  
- Retention labels  
- Trainable classifiers  

---

### Content Explorer
Allows viewing content containing:
- Sensitive information  
- Labels  

---

### Activity Explorer
Used to monitor:
- Label activity  
- User actions  
- Data-related events  

---

# 2. Protect Your Data

Protects information using labels, encryption, and access controls.

---

## Sensitivity Labels

Can be applied to:
- Documents  
- Emails  
- Calendar items  
- Teams meetings and chats  

---

## Protection Features

Includes:
- Encryption  
- Message encryption  
- File protection  
- SharePoint protection  

---

## Extended Labeling Support

Sensitivity labels can also be used with:
- File Explorer  
- PowerShell  

---

# 3. Prevent Data Loss

Integrates with:

- Microsoft Purview Data Loss Prevention (DLP)

Used to:
- Prevent accidental sharing  
- Protect sensitive information  
- Enforce compliance policies  

---

## Recommendations and Reports

Purview also provides:
- Security recommendations  
- Compliance insights  
- Reporting and analytics  

---

## Key Benefits

- Centralized data governance  
- Improved compliance management  
- Better visibility into sensitive information  
- Stronger protection against data leakage  

---

## Summary

Microsoft Purview is a comprehensive suite for data governance, protection, compliance, and risk management.

Microsoft Purview Information Protection specifically helps organizations:
- Discover sensitive data  
- Protect information using labels and encryption  
- Prevent data loss using integrated security controls  

---

## One-Line Summary

> Microsoft Purview helps organizations govern, protect, and manage data using classification, labeling, encryption, and compliance solutions.
