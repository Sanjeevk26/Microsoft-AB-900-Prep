# Microsoft Purview Activity Explorer

## Overview

Microsoft Purview Activity Explorer helps organizations monitor and investigate activities related to protected and labeled content.

It provides visibility into actions performed on content over the last 30 days using Microsoft 365 audit logs.

---

## Where to Access

Microsoft Purview Portal:

- Information Protection → Explorers → Activity Explorer  

---

## What is Activity Explorer?

Activity Explorer displays events and activities related to:

- Sensitivity labels  
- Retention labels  
- Data Loss Prevention (DLP)  
- Protected content  

It helps administrators understand:
- What actions occurred  
- When they happened  
- Who performed them  
- Which policies were involved  

---

# Data Sources Monitored

Activity Explorer can track activities across:

- Word documents  
- Excel files  
- PowerPoint presentations  
- Outlook emails  
- SharePoint  
- Exchange  
- OneDrive  

---

- Protection applied

- Protection removed

## File Access Activities

- Sensitive file read/opened  
- File discovered during scan

## Data Loss Prevention (DLP) Activities

- DLP policy match events

## Retention Label Activities

- Retention label applied  
- Retention label changed  
- Retention label removed

## File Save Activities

  - SharePoint  
  - OneDrive

# Common Columns Displayed

| Column | Description |
|--------|-------------|
| Activity | Action performed |
| File | File or content item |
| Location | Where activity occurred |
| Enforcement | Policy enforcement result |
| User | User associated with activity |
| Timestamp | When activity occurred |
| Policy | Related policy |
| Rule | Related rule |

# Customizing the View

You can:

- Customize displayed columns  
- Refresh activity data

---

# Viewing Activity Details

Selecting an activity provides additional information such as:
- Detailed event information  
- Related policies  
- Enforcement details  
- User and file context  

---

# Important Note

Some activities may occur automatically through:
- Policies  
- Auto-labeling  
- Copilot interactions  
- Automated compliance processes  

Not every activity is manually initiated by a user.

- Helps investigate compliance events  
- Tracks sensitive content usage  
- Supports auditing and governance
