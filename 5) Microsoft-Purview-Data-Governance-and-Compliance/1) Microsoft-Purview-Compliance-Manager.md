# Microsoft Purview Compliance Manager

## Overview

Microsoft Purview Compliance Manager helps organizations manage and monitor compliance across Microsoft Cloud services and other cloud environments.

It provides:
- Regulatory assessments  
- Compliance scoring  
- Improvement recommendations  
- Continuous monitoring and updates  

---

# Key Features

Microsoft Purview Compliance Manager includes:

- Multi-cloud regulatory assessments  
- Continuous control assessments  
- Regulatory updates  
- Common control mapping  
- Compliance scoring  

---

# Regulatory Updates

Compliance Manager automatically updates assessments when regulations or standards change.

This helps organizations stay aligned with evolving compliance requirements.

---

# Common Control Mapping

A single control or action can satisfy requirements across multiple regulations and standards.

This reduces duplication of effort and simplifies compliance management.

---

# Compliance Score

Compliance Manager provides a score indicating how well your organization aligns with selected standards and regulations.

> A high compliance score does NOT guarantee full compliance.  
> It indicates reduced risk of non-compliance.

---

# Accessing Compliance Manager

Microsoft Purview Portal:

- https://purview.microsoft.com

# Assessments

Assessments evaluate compliance against selected regulations and standards.

---

## Types of Assessments

### Pre-Built Assessments
Microsoft provides templates for:
- Industries  
- Countries  
- Regulations  
- Standards  

---

### Custom Assessments
Organizations can create tailored assessments.

---

# Creating an Assessment

When creating an assessment, you define:

- Regulation or standard  
- Assessment name  
- Assessment group  
- Applicable Microsoft services  

---

# Assessment Components

Assessments include:

| Component | Description |
|-----------|-------------|
| Microsoft Controls | Controls managed by Microsoft |
| Your Controls | Controls managed by your organization |
| Shared Controls | Responsibilities shared between Microsoft and the organization |

---

# Compliance Guidance

After assessments run, Compliance Manager provides:

- Step-by-step improvement guidance  
- Recommended actions  
- Risk reduction suggestions  

---

# Controls and Status Tracking

Controls are categorized by status:

- In Progress  
- Passed  
- Failed  
- Out of Scope  

---

# Improvement Actions

Compliance Manager tracks recommended improvement actions.

Examples:
- Security configuration updates  
- Policy implementation  
- Governance improvements  

---

## Action Status Examples

- Pending Update  
- In Progress  
- Completed

| Concept | Purpose |
|--------|---------|
| Compliance Score | Measures alignment with standards |

| Common Control Mapping | Reuse controls across frameworks |
| Improvement Actions | Recommended remediation tasks |
| Shared Controls | Compliance responsibilities shared with Microsoft |

---

# Important Note

A strong compliance score:
- Indicates improved security and reduced risk  
- Does NOT automatically mean legal or regulatory compliance  

Organizations must still validate compliance independently.
