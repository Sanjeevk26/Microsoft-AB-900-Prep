# Microsoft Entra Overview

## Introduction

Microsoft Entra is a suite of identity and access management solutions that helps organizations implement a **Zero Trust security model**.

### Key Benefits:
- Secure access to resources  
- Simplified user experience (sign-in, SSO)  
- Centralized identity and access management  
- Strengthened identity security using AI  

---

## What is Microsoft Entra?

Microsoft Entra is **not a single product**, but a collection of identity and security solutions.

---

## Core Microsoft Entra Products

### 1. Microsoft Entra ID
*(Previously Azure Active Directory)*

- Cloud-based identity and access management solution  
- Used by:
  - Microsoft 365  
  - Azure  
  - Dynamics 365  

**Features:**
- Multi-Factor Authentication (MFA)  
- Single Sign-On (SSO)  
- Identity management  

---

### 2. Microsoft Entra ID Protection
- Detects and reports identity-based risks  
- Provides risk-based insights for security  

---

### 3. Microsoft Entra ID Governance
- Automates:
  - Access requests  
  - Access assignments  
  - Access reviews  
- Uses lifecycle management  

---

### 4. Microsoft Entra External ID
- Enables secure access for external users  
- Supports:
  - B2B (Business-to-Business)  
  - B2C (Business-to-Consumer)  

---

## Additional Microsoft Entra Products

### Microsoft Entra Domain Services
- Supports legacy applications in the cloud  
- Enables older authentication methods  

---

### Microsoft Entra Workload ID
- Provides identity for:
  - Applications  
  - Services  
  - Containers  

---

### Microsoft Entra Private Access
- Secure access to corporate apps and networks  
- Works across multi-cloud environments  
- Eliminates need for VPN  

---

### Microsoft Entra Internet Access
- Secure internet and SaaS access  
- Protects Microsoft 365 and other cloud apps  

---

### Microsoft Entra Verified ID
- Credential verification using decentralized identity (DID)  
- Uses digital signatures on user devices  

---

## Pricing Overview

### Free Tier
- Included with:
  - Azure  
  - Microsoft 365  
  - Dynamics 365  
  - Intune  
  - Power Platform  

**Features:**
- MFA (app, SMS, phone call)  
- Self-Service Password Reset (SSPR)  
- Basic identity management  
- Trusted devices support  

---

### Plan P1 (~$6/user/month)

**Includes:**
- Microsoft Entra Verified ID  
- Microsoft Entra Internet Access  
- Partial:
  - Entra ID Protection  
  - Entra ID Governance  

**Additional Features:**
- Fraud alerts  
- MFA reports  
- Custom greetings and caller ID  
- Trusted IPs  
- MFA for on-prem apps  
- Conditional Access (basic, not risk-based)  
- SSPR with writeback (to on-prem AD)  

---

### Plan P2 (~$9/user/month)

**Includes everything in P1 +:**
- Full Entra ID Protection  
- Risk-based Conditional Access  
- Real-time sign-in risk analysis  
- Vulnerability detection  
- Risk event investigation  

**Also adds:**
- More Entra ID Governance features  
- Microsoft Entra Private Access  

---

### Microsoft Entra Suite (~$12/user/month add-on)

- Requires P1  
- Adds:
  - Full Entra ID Governance  
  - Advanced identity and access capabilities  

---

## Key Features Explained

### Multi-Factor Authentication (MFA)
- Adds a second layer of security:
  - Mobile app  
  - SMS  
  - Phone call  

---

### Conditional Access
- Controls access based on:
  - User  
  - Device  
  - Location  
  - Risk level  

---

### Self-Service Password Reset (SSPR)
- Allows users to reset passwords without admin help  
- Supports writeback to on-prem Active Directory

## Summary

- Microsoft Entra enables **Zero Trust security implementation**  
- It is a **suite of products**, not a single service  
- Core components:
  - Entra ID  
  - ID Protection  
  - Governance  
  - External ID  

> Microsoft Entra provides a centralized, AI-powered identity platform to secure access across users, devices, and applications.
