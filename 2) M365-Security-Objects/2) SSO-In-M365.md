# Single Sign-On (SSO) in Microsoft Entra

## Overview

Single Sign-On (SSO) is a feature in Microsoft Entra ID that allows users to access multiple independent applications by signing in **only once**.

After authenticating in one application, users can access other connected applications without needing to sign in again.

---

## What is Single Sign-On?

Single Sign-On enables:

- One-time authentication  
- Access to multiple systems without repeated logins  
- Seamless user experience across applications  

### Example Flow:
1. User signs in to Microsoft Entra / Azure  
2. Authentication is verified  
3. User gains access to multiple applications without re-entering credentials  

---

## Where SSO Can Be Used

- Cloud applications  
- On-premises applications  
- Custom-built applications  
- Hybrid environments (cloud + on-premises)  

---

## Benefits of Single Sign-On

### 1. Improved User Experience
- Users sign in only once  
- No need to remember multiple credentials  
- Reduced login friction  

---

### 2. Increased Productivity
- Faster access to applications  
- Less time spent on authentication  
- Fewer password-related interruptions  

---

### 3. Enhanced Security
- Reduces password reuse  
- Centralized authentication  
- Works with Multi-Factor Authentication (MFA)  

---

### 4. Centralized Management
- Users and access managed through Microsoft Entra  
- Easier administration of identities and permissions  

---

## Supported Applications

SSO works with:

- Microsoft 365  
- GitHub  
- Salesforce  
- Thousands of applications in the Microsoft Entra Application Gallery  

---

## Integration Capabilities

- Can be integrated with:
  - Custom applications  
  - On-premises Active Directory (Windows Server)  
- Supports hybrid identity scenarios  

---

## Cost

- Included with Microsoft Entra ID  
- No additional cost for basic SSO functionality  

---

## Summary

- SSO allows users to access multiple applications with a single login  
- Improves user experience and productivity  
- Enhances security by reducing password usage  
- Centralizes identity and access management
